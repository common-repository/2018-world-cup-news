Widget to display latest World Cup 2018 news

== Description ==
This plugin takes latest feed from fifa.com for Fifa  World Cup Russia 2018 and display the latest news in sidebar or footer area through wordpress widget and gives user option to refresh feed using ajax, only widget is refreshed not whole page. When clicked on picture the whole news with story will display on thick box


== Installation ==

1. Upload the folder `worldcup2018news` and its contents to the `/wp-content/plugins/` directory or use the wordpress plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new "World Cup 2018 News" will be available under Appearance > Widgets, where you can add it to your sidebar
4. Or search for the plugin through add plugin page on dashboard and install it
  


= Uninstall =

1. Deactivate World Cup 2018 News in the 'Plugins' menu in Wordpress.
2. After Deactivation a 'Delete' link appears below the plugin name, follow the link and confim with \'Yes, Delete these files\'.
3. This will delete all the plugin files from the server as well as erasing all options the plugin has stored in the database.
